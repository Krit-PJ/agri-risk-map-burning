# Print Header Final Refinement

ปรับปรุงล่าสุด:

- ปรับสีตัวอักษรปุ่มซ่อน/แสดงเมนูด้านซ้ายให้เป็นสีทึบ อ่านชัดเจน
- ปรับหัวรายงาน PDF ให้โลโก้ซ้าย/ขวา และชื่อรายงานอยู่ตรงกลาง
- ตัดบรรทัด “พัฒนาโดย...” ออกจากรายงานพิมพ์ เพื่อเพิ่มพื้นที่และลดความแน่น
- เพิ่มแถบข้อมูลด้านบนในรายงาน: ข้อมูลล่าสุด, วันที่พิมพ์รายงาน, ขอบเขต
- ปรับความกว้างรายงานให้อยู่ห่างจากขอบกระดาษ ไม่ชิดขอบขวา
- ปรับตารางเปรียบเทียบ Hotspot ให้มี safe margin และลดโอกาสข้อความถูกตัด
- คงแผนที่, ตาราง, donut และส่วนอื่นของระบบตามเดิม
