# Risk year and print contrast fix

การปรับปรุงเฉพาะจุด โดยคงส่วนอื่นของระบบเดิม:

1. คำนวณคะแนนความเสี่ยงใหม่ทุกครั้งเมื่อเปลี่ยนปี Hotspot หรือเปลี่ยนอำเภอ/ตำบล
2. ใช้ฐานอ้างอิง Hotspot สูงสุดย้อนหลังร่วมกันในแต่ละขอบเขต เพื่อให้สีสามารถเปลี่ยนตามปริมาณจริงระหว่างปี 2567, 2568 และ 2569 ไม่ใช่เปรียบเทียบอันดับภายในปีอย่างเดียว
3. เมื่อเลือกอำเภอ จะคำนวณและจัดระดับตำบลเฉพาะภายในอำเภอนั้น
4. แผนที่, Donut และ Top 10 ใช้ risk_score ชุดเดียวกัน
5. ก่อนพิมพ์ เปลี่ยนสีแกนและชื่อพื้นที่ในกราฟ Top 5 เป็นสีดำและตัวหนา แล้วคืนค่าสีเดิมหลังพิมพ์
