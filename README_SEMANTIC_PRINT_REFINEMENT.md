# Semantic / Print Refinement Update

ปรับปรุงคำเรียกชุดข้อมูลปี, ข้อความสะสมรายเดือน, สีตัวอักษรทึบ และรายงานพิมพ์ให้ตารางเปรียบเทียบ Hotspot กับระดับความเสี่ยงสะสมแสดงครบและอ่านง่ายขึ้น

## แก้ไขหลัก
- เปลี่ยนคำว่า `ปีประเมิน` เป็น `ชุดข้อมูลปี` เพื่อสื่อว่าเป็นปีของชุดข้อมูล Hotspot ที่เลือกใช้งาน
- เปลี่ยนสถานะ Timeline เป็น `จุดความร้อนสะสมรายเดือน ปี ....: ...`
- ปรับสีข้อความ ปุ่มพิมพ์รายงาน และองค์ประกอบสำคัญให้เป็นสีทึบ
- ปรับรายงานพิมพ์ A4 landscape ขอบ 1 ซม. ให้แผนที่อยู่ซ้าย และด้านขวาเป็นตารางเปรียบเทียบ Hotspot ตามด้วยระดับความเสี่ยงสะสม
- ซ่อน chart/ตารางที่ไม่จำเป็นในโหมดพิมพ์เพื่อป้องกันการทับซ้อนและหน้าว่าง
