# Executive readability and monthly cumulative UX update

ปรับปรุงตามข้อเสนอผู้ใช้งาน:

1. ปรับฟอนต์ สี และขนาดตัวอักษรให้เหมาะกับผู้บริหารและผู้ใช้งานสูงอายุ
   - ใช้ Sarabun / Noto Sans Thai
   - เพิ่มขนาดหัวข้อ ปุ่ม ตาราง และ subtitle
   - เพิ่ม contrast ของตัวอักษรและพื้นหลัง

2. ตัดส่วน "ช่วงเวลา Hotspot" ออกจาก Sidebar
   - เน้นการดู Hotspot สะสมรายเดือนผ่าน Timeline ด้านบน
   - ไม่แสดงตัวเลือกวันที่ใน Sidebar เพื่อไม่ให้ผู้ใช้สับสน

3. จัดเรียงปี พ.ศ. แบบซ้ายไปขวา
   - แสดงปุ่มปี 2566 → 2567 → 2568 → 2569
   - ปีประเมินหลักยังควบคุมข้อมูลทุกส่วนร่วมกัน

4. ปรับชื่อชั้นข้อมูล Burn Scar
   - เปลี่ยนจาก "Burn Scar (แผลไฟ)" เป็น "Burn Scar"

5. เพิ่มขนาดตัวอักษร subtitle
   - ข้อความ "ระบบประเมินและเฝ้าระวังความเสี่ยงการเผาในพื้นที่เกษตร จังหวัดกำแพงเพชร" ใหญ่ขึ้นและอ่านง่ายขึ้น
