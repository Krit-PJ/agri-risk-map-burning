# Print Table + Donut Refinement

ปรับรายงานพิมพ์ให้เน้นข้อมูลที่อ่านได้ครบ:

- ตัด Card เกณฑ์ระดับความเสี่ยงออกจากรายงานพิมพ์ เพื่อคืนพื้นที่ให้ตารางและ Donut chart
- ขยายพื้นที่ตารางเปรียบเทียบ Hotspot ด้านขวา
- คงตารางให้แสดงครบทุกคอลัมน์: พื้นที่, ปีก่อน, ปีที่เลือก, เพิ่ม/ลด
- ย้าย Donut chart ไปไว้ล่างสุดของด้านขวา และล็อกขนาดไม่ให้ซ้อนกับตาราง
- คงส่วนอื่นของระบบเดิมไว้

เหตุผล: รายงานสำหรับผู้บริหารควรเน้นผลการเปรียบเทียบและสถานการณ์สะสม มากกว่าเกณฑ์นิยามซึ่งซ้ำกับสีในแผนที่และสามารถดูจากคู่มือ/หน้าระบบได้
