# Boundary Data Folder

Place GeoJSON boundary files here:
- province.geojson     (77 provinces)
- district.geojson     (districts / อำเภอ)
- subdistrict.geojson  (subdistricts / ตำบล)

## Download Sources
- GADM Thailand: https://gadm.org/download_country.html (select Thailand)
  - Level 1 = Province → province.geojson
  - Level 2 = District  → district.geojson
  - Level 3 = Subdistrict → subdistrict.geojson
- DOPA Open Data: https://www.dopa.go.th/
- Thailand GIS Hub: https://data.go.th/
