# Admin import access

- ปุ่ม **Admin** แสดงในส่วนหัว
- ส่วน **นำเข้า Hotspot จาก Excel** ถูกซ่อนจากผู้ใช้ทั่วไป
- เมื่อเข้าสู่ระบบสำเร็จ ระบบจะแสดงส่วน Import และปุ่มดาวน์โหลด GeoJSON
- Session เก็บใน `sessionStorage` และหมดอายุหลัง 30 นาที
- กด “ออกจากระบบ” เพื่อซ่อนสิทธิ์ Import ทันที

## ข้อจำกัดด้านความปลอดภัย

เว็บไซต์นี้เผยแพร่บน GitHub Pages ซึ่งเป็น Static Hosting จึงไม่มีระบบยืนยันตัวตนฝั่งเซิร์ฟเวอร์ การตรวจสอบนี้เป็น client-side access gate เพื่อซ่อนและจำกัด UI เท่านั้น ไม่ใช่ระบบรักษาความปลอดภัยสำหรับข้อมูลสำคัญ ผู้ใช้ที่มีความรู้ด้าน Developer Tools อาจตรวจสอบหรือข้ามกลไกฝั่ง Browser ได้

สำหรับ Production ที่ต้องการความปลอดภัยจริง ควรย้ายการ Import ไปยัง Backend/API และใช้ OAuth, Firebase Authentication, Supabase Auth หรือระบบ SSO ของหน่วยงาน พร้อมตรวจสิทธิ์ฝั่งเซิร์ฟเวอร์
