# Print Report Stabilization Update

การปรับปรุงรอบนี้แก้ปัญหาหลักจากรายงาน PDF ที่ตารางด้านขวาถูกตัด กราฟ/legend อ่านยาก และ layout พิมพ์ยังไม่เสถียร

## สิ่งที่ปรับ

- ปรับแถบซ่อน/แสดงเมนูให้ตัวอักษรสีทึบ อ่านชัด
- ปรับ risk legend และหัวตารางบนหน้าเว็บให้ใช้พื้นขาว/ตัวอักษรทึบ
- ลดขนาดตารางเปรียบเทียบ Hotspot บนหน้าจอขวาเพื่อให้เห็นทุกคอลัมน์มากขึ้น
- ปรับรายงานพิมพ์ให้ใช้พื้นที่ปลอดภัย 270 มม. แทนการใช้เต็ม 277 มม. เพื่อลดการถูกตัดขอบจาก browser print
- ลดขนาดแผนที่เล็กน้อยและเพิ่มพื้นที่ให้ตารางด้านขวา
- ตัดเกณฑ์ระดับความเสี่ยงออกจากรายงานพิมพ์
- วาง Donut chart ไว้ล่างสุดและแยกจากตารางชัดเจน

## ข้อเสนอเชิงเทคนิค

Browser print ของแผนที่ Leaflet + Chart.js + ตาราง sticky มีข้อจำกัด หากต้องการรายงาน PDF ที่นิ่ง 100% ควรทำระบบ export PDF แยกโดยใช้ Playwright/Puppeteer ฝั่ง server หรือใช้ html2canvas + jsPDF โดยต้องจัดการ CORS ของแผนที่ฐานให้ดี
