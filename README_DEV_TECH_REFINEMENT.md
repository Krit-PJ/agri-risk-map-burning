# Dev Tech Refinement Update

ปรับปรุงรอบนี้เน้นความน่าเชื่อถือของหน้าจอและรายงานพิมพ์ โดยคงโครงสร้างระบบเดิมไว้

## สิ่งที่แก้ไข

1. ตัวนับผู้เข้าชม
   - ใช้ external counter ตามสคริปต์ที่กำหนด
   - เพิ่ม fallback แบบ localStorage เมื่อบริการภายนอกถูก browser/adblock/CSP บล็อก เพื่อไม่ให้ช่อง counter ว่าง

2. อัปเดตข้อมูลล่าสุด
   - เปลี่ยนจากวันที่ปัจจุบันเป็นวันที่ล่าสุดที่พบในชุดข้อมูล Hotspot ที่ระบบโหลดหรือข้อมูลที่นำเข้า

3. ปุ่มพิมพ์รายงานและอัปเดต
   - ใส่กรอบและจัดรูปแบบให้สอดคล้องกับปุ่ม Login

4. รีเซ็ตตัวกรอง
   - คงไว้เพราะมีประโยชน์ในการกลับสู่ค่าเริ่มต้นของพื้นที่ ปี เดือน และวันที่
   - แก้ให้ทำงานชัดเจนขึ้นและ dispatch filter-change

5. รายงานพิมพ์
   - ตั้ง A4 landscape margin 1 ซม.
   - เพิ่มหัวรายงานพร้อมโลโก้ ชื่อระบบ เครดิตผู้พัฒนา ข้อมูลล่าสุด และวันที่พิมพ์
   - แสดงปีประเมินและเดือนที่เลือก
   - จัดแผนที่ไว้กลางกรอบ ขวาเป็นตารางเปรียบเทียบ Hotspot และล่างเป็นระดับความเสี่ยงสะสม

6. กราฟและสี hover
   - ปรับสีตัวเลข/แกนกราฟให้เป็นสีทึบ
   - ปรับ hover card ให้สว่าง ไม่มืดทึบ
