# Sidebar + Comparison Table Readability Update

การปรับปรุงรอบนี้เน้นสองจุดตามผลการทดสอบหน้าเว็บจริง:

1. เพิ่ม/ปรับปุ่มซ่อน-แสดงเมนูซ้ายให้มองเห็นชัดเจนบน Desktop โดยวางเป็นปุ่มลอยด้านซ้ายเหนือ panel แทนการเบียดอยู่ใน header
2. ปรับตาราง “เปรียบเทียบ Hotspot” ฝั่งขวาให้ตัวอักษรเล็กลงพอดี ลดการเลื่อนแนวนอน และจัด column width ให้เห็นข้อมูลครบขึ้น

ยังคงโครงสร้างอื่นเหมือนเดิม และรวม print layout refinement ก่อนหน้าไว้ด้วย
