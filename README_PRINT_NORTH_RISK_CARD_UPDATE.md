# Print North Arrow and Risk Criteria Card Update

ปรับปรุงเฉพาะส่วนรายงานพิมพ์ PDF โดยคงส่วนอื่นของระบบไว้เหมือนเดิม

## สิ่งที่เพิ่ม

1. เพิ่มสัญลักษณ์ทิศเหนือไว้ด้านซ้ายบนของกรอบแผนที่ในโหมดพิมพ์
2. เพิ่ม Card เกณฑ์ระดับความเสี่ยง 4 ระดับไว้ด้านขวาล่างของรายงาน
3. ปรับ layout ด้านขวาของรายงานให้เรียงเป็น:
   - ตารางเปรียบเทียบ Hotspot
   - ระดับความเสี่ยงสะสม
   - เกณฑ์ระดับความเสี่ยง 4 ระดับ

## เหตุผล

- สัญลักษณ์ทิศเหนือช่วยให้รายงานที่พิมพ์ออกมาเป็นเอกสารแผนที่สมบูรณ์ขึ้น
- Card เกณฑ์ 4 ระดับช่วยให้ผู้บริหารตีความสีความเสี่ยงได้โดยไม่ต้องเปิดคู่มือประกอบ
- เก็บข้อมูลสำคัญไว้ในหน้า A4 แนวนอน โดยไม่เพิ่มภาระต่อหน้าจอใช้งานปกติ
