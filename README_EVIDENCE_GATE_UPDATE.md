# Annual Evidence Gate Update

## การเปลี่ยนแปลง

- เลือก Hotspot เพียง 1 ปี: ใช้โมเดลสถานการณ์รายปีแบบ Evidence Gate
  - 0-2 จุด = ต่ำ
  - 3-5 จุด = ปานกลาง
  - 6-10 จุด = สูง
  - มากกว่า 10 จุด = สูงมาก
- ชนิดพืชและขนาดพื้นที่ปรับคะแนนได้เฉพาะภายในระดับ ไม่สามารถยกระดับข้ามชั้น
- เลือก 2 ปีขึ้นไป: ใช้ Model A 40/20/20/20 ตามเดิม
- สีแผนที่, Donut, Top 10, Tooltip และรายงาน ใช้ risk_score ชุดเดียวกัน
- อัปเดตคู่มือ HTML/PDF ให้ตรงกับวิธีประเมินใหม่
