# Final UI refinement update

ปรับปรุงตามข้อเสนอผู้ใช้:

1. ปรับสีตัวอักษรให้เป็นสีทึบและเพิ่ม contrast ทุกส่วน
2. ตัดปุ่ม preset `ม.ค.–พ.ค.` และ `ประกาศห้ามเผา` ออกจาก Timeline
3. ใช้ `ปีประเมิน` เป็น Dropdown หลักเท่านั้น
4. ย้ายตัวเลือก `อำเภอ` และ `ตำบล` ไปไว้ใน `ชั้นข้อมูล / ขอบเขตข้อมูล`
5. ตารางเปรียบเทียบ Hotspot รายอำเภอแสดงครบ 11 อำเภอ โดยจัดคอลัมน์เป็น `อำเภอ | ปีก่อน | ปีที่เลือก | เพิ่ม/ลด`
6. คงโครงสร้างหลักของระบบเดิมไว้

ไฟล์หลักที่ปรับ:

- `index.html`
- `css/style.css`
- `js/app.js`
- `js/dashboard.js`
