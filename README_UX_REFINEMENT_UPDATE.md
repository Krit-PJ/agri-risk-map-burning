# UX Refinement Update

ปรับตามข้อสังเกตการใช้งานจริง:

1. ลดความซ้ำซ้อนของปี เหลือ Timeline เป็นตัวควบคุมปีประเมินหลักเพียงจุดเดียว
2. Timeline เดือนเลือกได้หลายเดือน เพื่อดู Hotspot สะสมตามช่วงประกาศห้ามเผา
3. Smartphone ใช้แนวทาง map-first และ panel ซ่อน/แสดงเป็น drawer
4. ปรับโทนสีเป็น Light Agri Dashboard สดใสขึ้น แต่ยังคงสีความเสี่ยง 4 ระดับ

ทุกส่วนใช้ filter state เดียวกัน: ปีประเมิน + เดือนที่เลือก + วันที่ + อำเภอ + ตำบล + ชนิดพืช
