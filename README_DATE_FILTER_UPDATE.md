# Date filter update

อัปเดตระบบจากไฟล์ `index.html` เวอร์ชันใช้งาน โดยเพิ่มตัวกรอง Hotspot แบบเชื่อมโยง **ปี → เดือน → วันที่**

## การทำงาน

- เลือกปีจากช่องกรอง จะเลือกชั้น Hotspot ของปีนั้นเพียงปีเดียวโดยอัตโนมัติ
- เลือก “ทุกปีที่เลือก” จะเปิดทุกปีที่มีในระบบ
- รายการวันที่จะแสดงเฉพาะวันที่ซึ่งมีข้อมูลจริงในเดือนและปีที่เลือก
- เดือนและวันที่กรองพร้อมกันในทุกส่วน ได้แก่ จุด Hotspot, Risk map, Summary cards, Trend, Top 5, Donut, Top 10, Tooltip และข้อความขอบเขตวิเคราะห์
- สูตร Risk ใช้เฉพาะ Hotspot ที่ผ่านตัวกรองปี เดือน วันที่ พื้นที่ และชนิดพืชเดียวกัน
- รองรับวันที่หลายรูปแบบในข้อมูลเดิม: `YYYY/MM/DD`, `YYYY-MM-DD`, ปี พ.ศ. และ Excel serial date
- การนำเข้าปีใหม่จะเพิ่มปีในตัวเลือกปีโดยอัตโนมัติ
