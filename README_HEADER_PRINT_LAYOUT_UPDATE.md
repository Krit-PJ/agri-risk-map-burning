# Header / Print Layout Refinement

ปรับปรุงจากข้อสังเกตล่าสุด:

1. ปรับแถบ Header ด้านขวา
   - กล่องผู้เข้าชมไม่ปล่อยให้ว่างเมื่อ external counter ถูก block
   - เพิ่ม fallback counter ภายในให้เห็นทันที
   - ปรับปุ่มพิมพ์รายงานให้มีข้อความชัดเจน ไม่เหลือแค่ไอคอน
   - ปรับขนาดและสีปุ่มให้อ่านง่ายขึ้น

2. ปรับรายงาน PDF
   - ซ่อน Top 10 และกราฟที่ไม่จำเป็นในโหมดพิมพ์ เพื่อลดการซ้อนทับ
   - จัดด้านขวาของรายงานเป็น 3 ส่วนชัดเจน:
     - ตารางเปรียบเทียบ Hotspot
     - ระดับความเสี่ยงสะสม
     - เกณฑ์ระดับความเสี่ยง 4 ระดับ
   - ลดขนาด North arrow ให้ไม่กินพื้นที่แผนที่
   - ควบคุม layout ให้พิมพ์ใน A4 แนวนอน 1 หน้าได้ดีขึ้น
