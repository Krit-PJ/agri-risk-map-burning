# UX Multi-month Timeline Update

ปรับปรุงจากเวอร์ชัน Timeline/Comparison/Print เดิม โดยคงโครงสร้างหลักของระบบไว้ และปรับเฉพาะ UX/Filtering/Presentation ตามข้อเสนอร่วมกัน

## สิ่งที่ปรับ

1. ลดความซ้ำซ้อนของการเลือกปี
   - ใช้ `ปีประเมิน` บน Timeline เป็นตัวเลือกหลัก
   - ซ่อน checkbox Hotspot รายปีใน Sidebar แต่ยังคงอยู่ใน DOM เพื่อให้ระบบเดิมทำงานร่วมกันได้
   - ปีประเมินจะกำหนดชั้น Hotspot และการคำนวณ Risk/Chart/Table ทั้งหมด

2. Timeline เดือนเลือกได้หลายเดือน
   - เลือกเดือนหลายเดือนแบบ Toggle
   - มี Preset: ทั้งปี, ม.ค.–พ.ค., ช่วงห้ามเผา, ล้าง
   - ทุกส่วนของระบบใช้ช่วงเดือนเดียวกัน
   - การเปรียบเทียบปีปัจจุบันกับปีก่อนใช้ช่วงเดือนเดียวกัน

3. ตารางเปรียบเทียบ Hotspot รายพื้นที่
   - เรียงคอลัมน์ปีจากเก่าไปใหม่: ปีก่อน → ปีที่เลือก
   - แสดงแถวรวมจังหวัด/รวมอำเภอไว้ด้านบนของตาราง
   - รายการย่อยเรียงตาม Hotspot ของปีประเมินจากมากไปน้อย

4. Smartphone Map-first
   - แผนที่เป็นพื้นที่หลัก
   - ปุ่มชั้นข้อมูล/สรุปผลเป็นปุ่มลอยด้านล่าง
   - Panel ซ้าย/ขวาเปิดแบบ Drawer และซ่อนได้

5. โทนสีสดใสขึ้น
   - เปลี่ยนเป็น Light Agri Dashboard
   - Header เขียวเกษตร/ฟ้าเขียว
   - Panel พื้นขาว เส้นขอบเขียวอ่อน
   - Risk color ใช้เขียว เหลือง ส้ม แดง ที่สดขึ้น
